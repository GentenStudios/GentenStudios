What is Genten Studios?

Genten Studios is a group of programmers who are passionate about creating games. All of our games follow a common pattern that the program itself does not provide any content. Content is defined by modules written in Lua which the game does the heavy lifting turning it into something playable. In the long-term our goal is to create Quartz Engine to be the core of the games we create in this style.

Project Phoenix

Our project coded "Phoenix" is the first game that we are developing in this style. Phoenix is an open world sandbox game where the player can manipulate anything in a cubic world around them. Check out our source code here.

Quartz Engine

Once Project Phoenix has hit an alpha state, the development of Quartz Engine is planned to begin. Quartz Engine will serve as a common library shared between Phoenix and future games we develop in this same style.
