What is Genten Studios?

Genten Studios is a group of creators who are passionate about creating games. All of our games follow a common pattern that the program itself does not provide any content. Content is defined by modules written in Lua which the game does the heavy lifting turning it into something playable. In the long-term our goal is to create Quartz Engine to be the core of the games we create with this design.

Project Phoenix

Our project coded "Phoenix" is the first game that we are developing with this design principle in mind. Phoenix is an open world sandbox game where the player can manipulate anything in a voxel world around them. Check out our source code here.

Quartz Engine

Once Project Phoenix has hit an alpha state, the development of Quartz Engine is planned to begin. Quartz Engine will serve as a common library shared between Phoenix and future games we develop following the same design pattern of content served by Lua modules.
